# r-real-estate-price

### 1. 주제 

  부동산 관련 기사가 부동산 가격에 어느정도 영향을 미치는지를 분석하기 위해 작성 되었습니다. 먼저 부동산 관련 기사의 헤드라인, 본문, 댓글 등에서 빈번히 등장하는 단어를 얻는다. 얻어진 단어들과 실제 부동산 가격과의 관계를 확인해본다.
